Прототип распознавателя номеров.
Запускается через файл ALPNR 2.0.py . В нём нужно передать путь к фотографии в качестве аргумента функции Recognizer.
Функция возвращает json с гос. номером в случае его нахождения.

1.jpg  фотография для примера, путь к которой можно передать функции Recognizer в качестве аргумента.
haarcascade_russian_plate_number.xml  каскадный классификатор Хаара.
